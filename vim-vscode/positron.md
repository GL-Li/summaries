## Outlines

### settings --> outline

- Disable the following outlines
  - variables: too many
  - constant
  - number
  - array
  - boolean
  - null
  - number

### R outlines like in RStudio

Enable `setting --> outline:show` string to RStudio style outlines in `xxx.R` file:

```r
# topic 1 ====================
## topic 2 --------
fff <- function() {
    "hellow world"
}
```
